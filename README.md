# MyWall
MyWall is a perfect alternative for blog app

# Tech Used
## Frontend
- Flutter
## Backend
- Firebase
## Database
- FirebaseAuth - Login and Registration
- FirebaseStorage - Storing Blog Images

## How to start this project
```bash
flutter clean
```
```bash
flutter pub get
```
```bash
flutter run
```
